## 📋 项目概述

YYC³ AILP (AI Learning Platform) 是一个基于人工智能的智能学习平台，旨在为用户提供个性化、智能化的学习体验。平台集成了先进的AI技术，包括智能推荐、自适应学习、实时评估等功能，为不同学习阶段和需求的用户提供精准的学习支持。

### 核心特性

- 🤖 **智能AI引擎**：基于多模态AI的智能学习助手
- 📊 **个性化推荐**：根据用户学习数据智能推荐学习内容
- 🎯 **自适应学习**：动态调整学习路径和难度
- 📱 **多端支持**：支持Web、移动端等多种设备
- 🔒 **安全可靠**：企业级安全架构和数据保护
- ⚡ **高性能**：优化的系统架构，确保流畅体验

## 🚀 快速开始

### 环境要求

- Node.js >= 18.x
- MySQL >= 8.0
- Redis >= 6.x (可选，用于缓存)

### 安装步骤

```bash
# 克隆项目
git clone https://github.com/YYC-Cube/learning-platform.git
cd learning-platform

# 安装依赖
pnpm install

# 配置环境变量
cp .env.example .env

# 启动开发服务器
pnpm dev
```
